- configurando com o github
git config user.name ""
git config user.email ""
gerar chave ssh e colocar no link (settings>keys)
selecionr o ssh do novo repositório
project options > git/svn version control system: git (reiniciar)
git remote -v (conferir se não tem nada)
git remote add origin [cópia do seu ssh do repo]
git remote -v (endereço deve aparecer)

boas práticas com o github (depois de configurado)
git pull
git add
git commit -m ""
git push
git status (conferindo como anda os adds pra dar apenas um commit)
